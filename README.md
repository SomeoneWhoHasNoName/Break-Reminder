# Break Reminder App 🚶‍♂️🌙

A beautiful, lightweight Windows desktop application designed to remind you to step away from your screen, stretch, and rest your eyes. 

Instead of a jarring, stressful alarm, this app uses calming, pure-CSS glassmorphism aesthetics and smooth animations to gently notify you when it's time for a break.

## ✨ Features

- **Unobtrusive System Tray Integration:** Runs silently in the background until it's time for your break.
- **Customizable Intervals:** Right-click the tray icon to easily adjust how often you want to be reminded (15 to 60 minutes) and how long your breaks should be.
- **Interactive Countdown:** An elegant full-screen countdown timer helps you actually commit to taking your full break.
- **Auto-Start:** Automatically launches quietly when you boot up your PC so you never forget to turn it on.
- **Zero Heavy Images:** The beautiful mountain and lake background is generated entirely with high-performance CSS code, making the app incredibly lightweight.

## 🚀 How to Install

You don't need to be a developer to use this!

1. Go to the **[Releases](../../releases)** page on the right side of this screen.
2. Download the `Break Reminder Setup 1.0.0.exe` file.
3. Double-click it to install, and the app will instantly appear in your system tray (down by your clock).

## 💻 Tech Stack
Built using **Electron**, **Node.js**, **HTML5**, and vanilla **CSS**. No bulky frameworks were used for the UI, ensuring maximum performance.
